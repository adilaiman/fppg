## 2 How to deploy on a local machine

### 2.1 Installing Meteor
Install meteor
```
https://www.meteor.com/
```

### 2.1 Running Meteor
```BASH
cd <cloned repo folder>
meteor npm install
meteor
```
If you still get package errors after running these commands you will need to run
```BASH
meteor npm rebuild
```
